ArduinoSensor enthält den Code um den Sensor auszuwerten und die Daten per UDP zu verschicken. 

Um die Datei bearbeiten zu können, wird ein Editor von Arduino benötigt mit welchem man den Code auf das Board flashen kann.
Unter folgender Adresse lässt sich der Editor herunterladen:

https://www.arduino.cc/en/Main/Software

Libarys lassen sich ebenfalls im Editor unter Sketch->Include Libary einbinden. 


Beispiele oder Codeschnipsel lassen sich hier finden:

https://www.arduino.cc/en/Tutorial/HomePage
